Implicit Casting (Widening Conversion), Explicit Casting (Narrowing Conversion)

Upcasting, Downcasting

Implicit casting adalah konversi otomatis yang dilakukan Java ketika mengubah tipe data dari ukuran kecil ke ukuran lebih besar, atau dari subclass ke superclass.
Tidak membutuhkan tanda cast ( ).
Contoh:
int a = 10;
double b = a;   // implicit

Explicit casting adalah konversi yang harus ditulis secara manual, biasanya dari tipe lebih besar ke tipe lebih kecil, atau dari superclass ke subclass.
Harus memakai tanda ( ).
Contoh:
double a = 10.5;
int b = (int) a;   // explicit
