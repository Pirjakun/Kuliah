1. Jelaskan JavaScript itu apa
   JavaScript adalah bahasa pemrograman untuk membuat halaman web interaktif dan dinamis.

2. Jelaskan jQuery itu apa
   jQuery adalah library JavaScript yang mempermudah penulisan kode dengan sintaks lebih singkat dan mudah.

3. Bagaimana cara assign elemen yang memiliki id dan cara menggunakan listener-nya
   Elemen dengan id diambil menggunakan `document.getElementById("id")` dan listener-nya menggunakan `addEventListener("event", fungsi)`.
   Contoh: `document.getElementById("btn").addEventListener("click", fungsiKlik)`

4. Jelaskan kegunaan JavaScript dan jQuery
   JavaScript digunakan untuk logika dan interaktivitas web, sedangkan jQuery digunakan untuk mempermudah manipulasi elemen dan event di JavaScript.
